# passcode-gen

Generates strong and complex passwords, has the ability to customize the length of the password you want.
Options when you want to generate a password

- Letters, ABCDEFG
- Digits, 123567890
- Symbols, ^@?"{}&\*
- Length, minimum 8 characters. Maximum, well. Unlimited.
